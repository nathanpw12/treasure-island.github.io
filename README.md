# 🏴‍☠️ Treasure Island

Welcome to **Treasure Island**! This is a classic text-based "Choose Your Own Adventure" game written entirely in **Python**. Your mission is simple: navigate through the dangerous paths, make the right choices, and find the hidden treasure. 

One wrong step, and it's **Game Over**! ☠️

## 🎮 Play the Game
You don't need to install Python to play! Thanks to **PyScript**, the game runs directly in your browser.

**[▶️ Click here to play Treasure Island](https://nathanpw12.github.io/treasure-island/)**

## 🛠️ Features
* **Interactive Storytelling:** Multiple paths and different endings based on your inputs.
* **ASCII Art:** Custom terminal graphics to bring the pirate theme to life.
* **Web-Based Terminal:** Built with an embedded Python engine running directly on HTML/CSS.
* **Dark Mode:** Styled with a retro hacker/terminal visual aesthetic.

## 💻 Tech Stack
* **Python 3:** Core game logic (nested conditionals and string manipulation).
* **PyScript:** To run the Python code natively inside the browser.
* **HTML/CSS:** To create the dark-themed web terminal interface.
